# Bem vindo(a) ao meu repositório ✌

Aplicação web desenvolvida em **ReactJS** que simula um fluxo básico de e-commerce. O sistema permite visualizar produtos, adicioná-los ao carrinho e finalizar a compra por meio de um formulário de checkout.

O projeto utiliza Redux para gerenciamento de estado, Redux-Form para controle de formulários e React Router para navegação entre páginas, além de Material-UI (MUI) para construção da interface.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

- **ReactJS**
- **Material-UI (MUI)**
- **JavaScript**
- **Redux**
- **Redux-Form**
- **React Router**

## Configuração do Projeto

Siga os passos abaixo para executar o projeto:

### 1. Clonar o repositório

```bash
git clone https://github.com/guilherme-mattos-conde/react-redux-ecommerce.git
```

### 2. Acessar a pasta do projeto

```bash
cd react-redux-ecommerce
```

### 3. Instalar as dependências

```bash
npm install
```

### 4. Executar o projeto

```bash
npm run dev
```
